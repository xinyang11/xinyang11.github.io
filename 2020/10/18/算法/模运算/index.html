模运算

基本规则

模运算的基本规则与四则运算基本一致：

  • (a + b) % p = (a % p + b % p) % p

  • (a – b) % p = (a % p – b % p) % p

  • (a b) % p = (a % p b % p) % p

  • (a^b) % p = ((a % p)^b) % p

这里没有除法的模运算是因为除法取模较为特殊。
